# Old_VRChat_Models_Vrc
旧vrchat模型
这是未完成的vrchat模型混合器文件，没有所有的混合形状和其他修改，它使用统一的表达式。这适用于任何可能需要基础模型以实现更快开发的项目，并且可以免费使用，无需信用，特别是对于vrchat。要使用该模型，您可能需要修复vrchat模型的一些错误，如果您想在vrchat中轻松使用该模型的话，您需要vrchat创建者compainion（VCC）和vrchat，并使用vrchat帐户而不是steam帐户。我使用的搅拌机版本是2.93，当前日期是2026年初。在VRChat Creator Companion中，我的VRChat SDK基础是3.10.1，VRChat的SDK化身是3.10.1。我正在使用vrm转换器与univrm进行vrchat。化身正在使用mtoon着色器。我有搅拌机中的CATS插件用于优化，还有搅拌机的vrm导入器插件。我在面部网格中添加了一些混合形状，但没有全部配置。
This is an unfinished VRChat model blender file, without all the blendshapes and other modifications, it uses a unified expression. This applies to any project that may require a basic model for faster development, and can be used for free without credit, especially for VRChat. To use this model, you may need to fix some errors in the VRChat model. If you want to easily use this model in VRChat, you need the VRChat creator Companion (VCC) and VRChat, and use a VRChat account instead of a Steam account. The mixer version I am using is 2.93, and the current date is early 2026. In VRChat Creator Companion, my VRChat SDK base is 3.10.1, and the VRChat SDK avatar is 3.10.1. I am using a VRM converter to engage in VRchat with UniVRM. The avatar is using the mtoon shader. I have the CATS plugin in the blender for optimization, as well as the VRM importer plugin for the blender. I added some mixed shapes to the facial mesh, but not all of them were configured.
